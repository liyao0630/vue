引入插件
在调用Vue.use时，第一个参数为：插件对象，第二个参数为：传递插件的数据；会调用插件install方法
插件install方法接受两个参数：第一个参数： vm === Vue， 第二个参数：Vue.use的第二个参数。 
    
      // 插件
      exprot default {
        install:function(vm, options){
          vm.prototype.$local ={
            show(){
              console.log('自定义插件的show方法');
            }
          }
        }
      }

      import utils from 'utils';
      Vue.use(utils , {a:1})
